  <h3 align="center">Stanford ENERGY260 Class Final Project</h3>

  <p align="center">
Latin Hypercube Sampling is found to be superior to simple random sampling in Monte Carlo simulations when using both Gaussian and exponential probability distribution functions. Experimental results on given examples and exercises matched the theoretically derived literature convergence rates. LHS was also applied to enhance the process of building proxy function using support vector machines with Gaussian kernel, and showed improved results in some cases and almost neutral results when compared to random sampling. Furthermore, LHS showed an improved performance in artificial neural network training efficiency when higher regression fit results for a given sample size. This eventually facilitates the applicability of surrogate-based optimization approach
